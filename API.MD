# API Documentation

## Global Schemas

For this project we will be using schemas from [schema.org](https://schema.org):

* [Place](https://schema.org/Place)
* [Event](https://schema.org/Event)
* [Article](https://schema.org/Article)

## API Documentation

The main API documentation is made using [OpenAPI 3](https://github.com/OAI/OpenAPI-Specification/blob/master/versions/3.1.0.md) ( [Swagger](https://swagger.io/resources/open-api/) ).
